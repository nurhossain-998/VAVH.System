How to Connect Google Sheets

Follow these steps — takes about 5 minutes

1

Create a Google Sheet

Go to sheets.google.com → create a new spreadsheet → name it "Vavh Finance". Copy the Spreadsheet ID from the URL.

2

Enable Google Sheets API

Go to Google Cloud Console → enable Google Sheets API.

3

Get OAuth 2.0 Access Token

In Cloud Console → APIs & Services → Credentials → Create OAuth 2.0 Client ID → Use OAuth Playground → scope: https://www.googleapis.com/auth/spreadsheets

4

Paste credentials below & Test

Enter your Spreadsheet ID and Access Token, click Test Connection, then Save Settings.
